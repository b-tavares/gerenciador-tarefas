#Gerenciador de Tarefas

##Objetivo

Criar uma aplicação MVC fullstack utilizando framework PHP. Objetivo final: C.R.U.D. de tarefas. Deve ser um gerenciador de tarefas e conter: 
- Nome;
- Prioridade;
- Prazo;
- Descrição;
- Pessoa.

##Features
- Cadastro de tarefas ✔
- Edição de tarefas ✔
- Exclusão de tarefas ✔

##Tecnologias
- PHP
- Blade
- Bootstrap
