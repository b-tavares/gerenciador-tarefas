<h1 align="center"><a>Gerenciador de Tarefas</h1>

<h2>Objetivo</h2>

Criar uma aplicação MVC fullstack utilizando framework PHP. Objetivo final: C.R.U.D. de tarefas. Deve ser um gerenciador de tarefas e conter: 
- Nome;
- Prioridade;
- Prazo;
- Descrição;
- Pessoa.

<h2>Features</h2>
* Cadastro de tarefas ✔
* Edição de tarefas ✔
* Exclusão de tarefas ✔

<h2>Tecnologias</h2>
- PHP
- Blade
- Bootstrap
